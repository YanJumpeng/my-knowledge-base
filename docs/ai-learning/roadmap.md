# AI学习路线图

这是我为自己设计的 12 个月 AI 学习路线，也分享给想转型的朋友。

---

## 🎯 总体规划

### 时间线
- **阶段一**：基础夯实（Month 1-2）
- **阶段二**：AI应用开发（Month 3-5）
- **阶段三**：产品化（Month 6-8）
- **阶段四**：规模化（Month 9-12）

## 📚 阶段一：基础夯实（Month 1-2）

### 目标
掌握 Python 进阶知识，熟练使用 AI 工具

### 学习内容
- [x] Python 基础复习
  - 数据类型、函数、类
  - 文件处理和异常处理
  - 常用库：requests, json, csv
- [ ] API 调用基础
  - HTTP 请求原理
  - 使用 requests 库
  - 认证和错误处理
- [ ] AI API 使用
  - GPT API 调用
  - Claude API 调用
  - Prompt 工程基础
- [ ] 命令行和 Git 基础

### 练习项目
1. ✅ **文章总结工具**：调用 AI API 自动总结长文章
2. 🔄 **数据爬虫**：爬取某个网站数据并保存
3. ⏳ **简单聊天机器人**：基于 AI API 的对话程序

### 学习资源
- 📖 Python官方文档
- 🎥 B站 Python 教程
- 📚 OpenAI / Anthropic API 文档
- 💬 AI辅助学习（ChatGPT/Claude）

### 每周时间分配
- **理论学习**：3 小时
- **代码练习**：8 小时
- **项目实战**：6 小时
- **总结记录**：3 小时

## 🚀 阶段二：AI应用开发（Month 3-5）

### 目标
能独立开发 AI 驱动的小工具和应用

### 学习内容
- [ ] LangChain 框架
  - Chain 的概念和使用
  - Memory 机制
  - Agent 基础
- [ ] 向量数据库
  - Embeddings 原理
  - Pinecone / Chroma 使用
  - 相似度搜索
- [ ] RAG（检索增强生成）
  - 文档加载和切分
  - 向量存储
  - 查询和回答
- [ ] FastAPI 基础
  - 路由和请求处理
  - 数据验证
  - API 文档

### 练习项目
1. **个人知识库助手**
   - 能回答我所有笔记的问题
   - 支持文档上传和向量化
   - 基于 RAG 的问答系统

2. **自动化内容生产工具**
   - 输入主题，自动生成大纲
   - AI 辅助写作和改写
   - 批量生成社交媒体内容

3. **垂直领域 AI 助手**
   - 针对某个具体行业（如加密）
   - 整合多个数据源
   - 提供专业建议

### 学习资源
- 📖 LangChain 官方文档
- 🎥 LangChain 实战教程
- 📚 《Building LLM Apps》
- 💻 GitHub 开源项目学习

## 💼 阶段三：产品化（Month 6-8）

### 目标
做出一个可以卖钱的 AI 产品

### 学习内容
- [ ] Web 开发
  - 前端：HTML/CSS/JavaScript 基础
  - 前端框架：React 入门
  - 状态管理和路由
- [ ] 完整的全栈开发
  - FastAPI + React 整合
  - 数据库设计（SQLite/PostgreSQL）
  - 用户认证和授权
- [ ] 产品思维
  - 需求调研和分析
  - MVP 设计
  - 用户体验优化
- [ ] 部署和运维
  - Vercel / Railway 部署
  - 域名和 HTTPS
  - 监控和日志

### 里程碑项目：做一个完整产品

**产品想法（选其一深度开发）：**

1. **媒体内容工作流系统**
   - 选题挖掘 → 资料收集 → 大纲生成 → 初稿 → 改写 → 发布
   - 适合自媒体和内容创作者
   - 定价：99-299元/月

2. **加密研报生成器**
   - 输入项目名称，自动抓取信息
   - 生成结构化研报
   - 适合投研人员
   - 定价：299-499元/月

3. **AI 批量处理工具集**
   - 图片、文字、数据批量处理
   - 多个小工具的集合
   - 适合小企业和个人
   - 定价：9.9-49元/月

### 产品开发流程
1. **Week 1-2**：需求调研，设计原型
2. **Week 3-6**：开发 MVP
3. **Week 7-8**：测试和优化
4. **Week 9-10**：找 10 个测试用户
5. **Week 11-12**：根据反馈迭代

### 变现准备
- 定价策略设计
- 支付系统集成
- 营销文案准备
- 推广渠道规划

## 💡 学习原则

### 1. 项目驱动
- 不为学而学，而是为了做项目
- 每学一个知识点就用项目巩固
- 项目从简单到复杂，逐步提升

### 2. 持续输出
- 每周至少 2 次公开更新
- 通过输出倒逼输入
- 建立反馈循环

### 3. 社群学习
- 加入学习社群
- 主动分享经验
- 互相监督和鼓励

### 4. 快速迭代
- 不追求完美，先完成再完美
- 快速试错，及时调整
- 保持灵活性
