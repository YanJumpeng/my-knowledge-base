# AI时代技术决策速查指南

> **核心理念**：不是精通代码，而是知道什么时候用什么工具

---

## 🎯 快速决策框架

当你要做一个功能/项目时，问自己3个问题：

1. **这是什么类型的任务？** → 决定用什么语言
2. **数据需要保存吗？** → 决定是否需要数据库
3. **用户在哪里使用？** → 决定前端形式

## 🔧 按任务类型选技术

### 1️⃣ 数据处理类

**场景**：
- 处理Excel/CSV文件
- 数据清洗和分析
- 批量处理文本

**技术栈**：
```
Python + Pandas + Jupyter Notebook
```

**为什么**：
- Python的Pandas专门处理数据
- Jupyter可以边写边看结果
- 库多，什么数据都能处理

**不需要学**：前端、数据库、复杂算法

**AI提示词模板**：
```
"我有一个CSV文件，包含[描述数据]，
我想要[具体需求]，
用Python和Pandas帮我写代码"
```

### 3️⃣ Web工具类（给自己用）

**场景**：
- 快速做个工具界面
- 数据可视化
- 内部使用的小工具

**技术栈**：
```
Python + Streamlit 或 Gradio
```

**为什么**：
- 几行代码就能做出界面
- 不用写HTML/CSS/JS
- 自动处理布局和交互

**不需要学**：React、Vue、复杂前端

**代码示例**：
```python
import streamlit as st

st.title("我的AI工具")
user_input = st.text_input("输入内容")
if st.button("处理"):
    result = process(user_input)  # 你的处理逻辑
    st.write(result)
```

**AI提示词模板**：
```
"用Streamlit做一个[工具名称]，
功能包括[功能1、功能2]，
界面要有[输入框/按钮/显示区]"
```

## 🗄️ 数据存储决策树

### 需要存数据吗？

**👉 不需要（大部分情况）**
- 用变量/字典存在内存
- 用JSON/CSV存文件
- **够用就行！**

**👉 需要，但数据量小（<10万条）**
- 用 **SQLite**
- 单个文件，不用安装
- Python内置支持

```python
import sqlite3
conn = sqlite3.connect('data.db')
# 然后用SQL操作
```

**👉 需要，数据量大或多用户**
- 用 **PostgreSQL**
- 更强大、更稳定
- 支持并发访问

**👉 需要，数据结构复杂/不固定**
- 用 **MongoDB**（NoSQL）
- 但大部分情况不需要！

## 🎨 前端技术速查

### 你需要知道的（5分钟理解）

**HTML**：网页的骨架
```html
<div>这是一个盒子</div>
<button>这是一个按钮</button>
```

**CSS**：网页的样式
```css
.box { 
  color: blue; 
  font-size: 20px; 
}
```

**JavaScript**：网页的交互
```javascript
button.onclick = function() {
  alert("点击了！");
}
```

**React**：组件化开发（最流行）
```jsx
function MyButton() {
  return <button>点我</button>
}
```

### 你的策略：
1. **不深入学**，只了解基本概念
2. **用组件库**（shadcn/ui、Material-UI）
3. **让AI写代码**，你只需要改改
4. **够用就行**，不追求完美

存储：不需要（每次实时处理）

```

**开发时间**：半天

**AI提示词**：
```
用Python和Streamlit做一个文章总结工具
功能：
1. 文本框输入文章
2. 调用OpenAI API生成摘要
3. 显示结果
请给出完整代码
```

### 案例2：做一个加密项目研报生成器

**需求分析**：
- 输入：项目名称

  - 生成结构化研报

- 用户：自己和团队

**技术决策**：
```
语言：Python

**分步骤让AI帮你**：
```

```
